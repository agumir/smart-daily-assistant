# Smart Daily Assistant Agent

[![Deployed on Vercel](https://img.shields.io/badge/Deployed%20on-Vercel-black)](https://smart-daily-assistant.vercel.app)
[![Telegram Bot](https://img.shields.io/badge/Telegram-@SmartDailyAssistantBot-blue)](https://t.me/SmartDailyAssistantBot)

An AI-powered daily assistant available on **Web** and **Telegram** that helps users organize tasks, prioritize them, and create simple action plans.

## 🎯 Features

- ✅ **Task Extraction** - Automatically extract tasks from natural language
- ✅ **Priority Assignment** - High/Medium/Low priority based on urgency
- ✅ **Action Plan Generation** - Create step-by-step plans
- ✅ **Clarifying Questions** - Ask for missing information (deadlines, specifics)
- ✅ **Multi-channel Support** - Same AI logic powers Web + Telegram
- ✅ **Secure API Integration** - All keys on server-side only
